#  E-Commerce Sales Analysis Dashboard | Power BI, DAX

Developed a **multi-page interactive Power BI dashboard** to analyze e-commerce sales data and generate actionable business insights across sales, operations, and product performance.

### 🔹 Overview Dashboard
![Amazon sales Dashboard](amazon.png)

### 🔹 Key Contributions

* Designed and implemented **KPIs** including Total Sales, Total Orders, and Average Order Value using DAX
* Performed **time-series analysis** to identify sales trends, growth patterns, and seasonal fluctuations
* Analyzed **product performance**, identifying top categories and Top-10 SKUs contributing to revenue
* Evaluated **operational efficiency** using courier status and fulfillment insights
* Built **geographical analysis (city/state level)** to understand regional sales distribution

### 🔹 Dashboard Features

* Interactive dashboards with **filters (date, category, state)**
* **Sales Over Time (Line Chart)** for trend analysis
* **Category & SKU analysis** for revenue contribution
* **Operations dashboard** for delivery and fulfillment tracking
* **Map visualization** for location-based insights

### 🔹 Key Insights

* Identified **high-revenue product categories** driving majority of sales
* Observed **seasonal trends and fluctuations** in monthly sales
* Detected **delivery inefficiencies** through courier status analysis
* Applied **Pareto principle (Top 20% SKUs generating major revenue)**

### 🔹 Tools & Technologies

Power BI • DAX • Data Cleaning • Data Visualization
